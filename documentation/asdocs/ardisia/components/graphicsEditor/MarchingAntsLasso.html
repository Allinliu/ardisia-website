Classpublic class MarchingAntsLasso
InheritanceMarchingAntsLasso spark.components.Group

Displays a marching ants lasso over the passed bitmap data.
    DESIGN NOTES
  • Massively improve performance by decreasing the area that is masked to only the area that is visible to the user.
  • Performance problems stem from the fact that all regions that are not visible that are masked still consume processing power.
