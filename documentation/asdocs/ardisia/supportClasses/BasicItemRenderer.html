Subclasses WheelListBaseRenderer

A simple item renderer designed to be overridden by more complex renderers. Much lighter weight than the ItemRenderer included with the Flex SDK.

Call invalidateRendererState() to update the state. Override getCurrentRendererState() to update what state is set when the state is updated. These methods provide an efficient method to update a renderer's state.

Adds support for a "down" and "hovered" state in addition to the included Flex SDK IItemRenderer interface.

Note: many of the IItemRenderer properties are not implemented by default like the caret or dragging. Override to add support.
